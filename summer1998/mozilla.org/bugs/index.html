bugs

Welcome to the mozilla.org bug center!

  • Are you a developer who has a bug to report? If so, please visit the mozilla.org bug reporting page.

  • Are you a user who has a bug to report? If so, please report the bug to the person or organization who provided you with the executable program in which you found the bug. For example, if you want to report a bug in the Netscape Communicator end-user software, you're in the wrong place: please head on over to netscape.com instead.

  • Are you a developer who would like to browse the database of bugs reported to mozilla.org? If so, please visit the mozilla.org bug query page.

  • Are you a user looking for technical support? Again, you'll need to consult the provider of the executable that you're running. If you're running Netscape Communicator, please head on over to netscape.com.

  • Do you want to look at the source for our bug system, and maybe use it for your own bugsystem? If so, check out the Bugzilla source page.

If you're a developer, and you're not sure whether some behavior you're seeing is a bug, you might want to discuss it with the relevant module owner, or in the appropriate newsgroup or mailing list.
