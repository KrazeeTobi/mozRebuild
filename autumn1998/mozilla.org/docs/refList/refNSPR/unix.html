Building NSPR 2.0 on Unix

4 March 1998 by Wan-Teh Chang (wtc@netscape.com)

First, set your CVSROOT environment variable to the appropriate CVS repository and check out the ns/nspr20 module.

    % cvs co ns/nspr20
NSPR is now self-contained in the ns/nspr20 module. It does not depend on any other directories under ns, including ns/config.

Note: If you want to build on Unixware, you also need to check out ns/build, because we need the hcc shell script there.

    % cvs co ns/build
Now you can build nspr20 from the root of the ns/nspr20 module.
    % cd ns/nspr20
    % gmake export
The first time you do gmake export, ns/dist will be created. Header files and libraries are copied into the platform-specific object directory under ns/dist.
