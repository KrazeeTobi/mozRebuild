projects

The primary mission of mozilla.org is to coordinate and integrate the work of others. Here are all of the ongoing projects of which we are aware.

Module Owners
The Module Owners document lists the points-of-contact for the various source code directories that comprise Mozilla: that is, the people responsible for the code that mozilla.org distributes. Talk to these people when you want to change the code for which they are responsible.

Blue Sky
The Blue Sky column is a pulpit whereby we (or rather, the editors of that column) advocate projects that nobody has yet started working on. If you are itching to work on something new, this is the place to go seeking inspiration. Much of the content of this column comes from and is inspired by the mozilla-wishlist forum.

New Layout
The New Layout project is a next-generation layout engine intended to eventually replace the current Mozilla layout engine. (This project was recently known as ``Raptor.'')

Mail/News
The Mail/News project is currently concerned with releasing and re-integrating the source code of Netscape Messenger, the mail/news reader.

Ports
Many different groups are planning ports of Mozilla to new platforms and toolkits. The Ports page list all of the known, active porting projects.

Other
This page lists links to other Mozilla-related projects of which we are aware, but which are not currently a part of Mozilla itself.
