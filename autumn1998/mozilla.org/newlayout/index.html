Next Generation
ayout
Lying at the heart of Mozilla 5.0, NGLayout is Mozilla's next generation, high performance layout engine based on open Internet standards such as HTML 4.0, CSS 1/2, XML 1.0, and the Document Object Model. Built using a modular, XPCOM-based architecture, NGLayout is an embeddable component. Your help is needed to make NGLayout the best layout engine available!

For the good of the code...
Want to help implement NGLayout's W3C DOM support? Check out our DOM Projects page for info on how to get involved today.
