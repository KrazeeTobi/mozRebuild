Team Resources

Mailing lists

The development team mailing list and mirrored newsgroups are:

mozilla-rhapsody@mozilla.org (subscribe/unsubscribe)
news://netscape.public.mozilla.rhapsody
news://news.mozilla.org/netscape.public.mozilla.rhapsody

If you would like to become a part of the team, please consider your ability to contribute along these lines:

  • Objective-C programming experience.
  • C/C++ experience; ObjC++ bridging knowledge.
  • NeXTStep/OpenStep/Yellow experience.
  • You must have a Rhapsody developer's licence.

We don't have an announcements list; please don't ask to be added to the developer's list in order to get updates. There won't be anything interesting for a while. Check these web pages for updates.

The source code

Source will be managed via client/server CVS, which will be provided by mozilla.org.
Details about how to get the source are described here. The repository is read-only,
developers with check-in privileges will be designated by module owners.
Code is also online here.
