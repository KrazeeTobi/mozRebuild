Browser and Composer Test Plans? What would I use these for?

The browser/composer front-end QA team publishes our test plans because we don't know everything.

Please feel encouraged to review the test plans presented below, and to contact any test plan authors with cogent feedback on their test plans. We welcome your thoughts on errors and major omissions --- particularly if you're willing to personally revise the test plan to address any deficiencies you identify.

What test plans are currently available?

Browser:

  • Bookmarks
  • Copy & Paste
  • Drag & Drop
  • History
  • Imaging
  • Text Selection
  • Toolbars
  • Wallet


    Composer:

  • User Interface:
    • Toolbar
    • Dialogs
    • Menus
  • Find & Replace
  • Selection of:
    • Images
    • Links
    • Text
    • Tables
  • Keyboard Navigation
  • Insertion of:
    • Horizontal Rule
    • Image
    • Link
    • Table
    • Target
  • Event Handling
  • Spell Checking
  • Table Editing
  • Editing Functions
  • Preferences
