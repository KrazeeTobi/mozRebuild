Table Generator for URL-specific Field-to-Schema Mapping

This utility allows you to automatically create a mapping for forms at a URL of your choice and to have that mapping transmitted to Netscape for inclusion in the master table that is being maintained on their server.  To use the utility, perform the following steps:

1. Enter a URL in the textbox in the frame above.  It must be a full URL -- i.e., it must start off with http:// or some equivalent protocol.  Note: this need not be the URL of the page containing the desired form but could instead be the URL of a page from which you can navigate to get to that form.

2. Press the "Load URL" button.  The page at the URL you just entered will appear in place of these instructions.  And a list of the select-lists contained on the page will appear in the frame at the right.

3. Navigate to the desired form if necessary.

4. In each text field, enter the corresponding schema name.  In other words, if the form contains text fields such as:

Name (first, middle, last):
you would enter the following:
Name (first, middle, last):
A list of the schema names to be used can be found here.

5. Press the "Transmit Data" button.  This will transmit your values to Netscape and, after being reviewed, will be included in Netscape's master table.
