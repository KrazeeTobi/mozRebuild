The newsgroup for discussing Mozilla editor issues is netscape.public.mozilla.editor.
It is gatewayed into the mailing list mozilla-editor; subscribe here (put "subscribe" as the subject of the message).

Preliminary design ideas for the new editor

The new editor will be based on NGLayout, which means that we will be rewriting everything, since the old editor was heavily intwined into the old layout engine.

Information on Composer in the old Mozilla codebase (pre-NGlayout); this code is no longer being developed or maintained, but is still in the tree for those who might want to reference it.

We are only beginning the design of the new editor. Specs here are very preliminary; feel free to comment on them in the editor newsgroup/mailing list.
