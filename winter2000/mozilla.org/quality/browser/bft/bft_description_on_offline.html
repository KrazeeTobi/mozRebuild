Basic Functional Test Case Description - Online/Offline

Initial Conditions:

Empty the disk and memory cache before you begin each test. You can find the cache section in the Advanced section of the Preferences dialog.

Description:

To verify switching between online and offline mode.

Switching between Modes from the Offline Submenu

  1. Launch Navigator.
  2. From the File menu, select Offline and then select Work Offline. The status bar should display the message "You are offline."
  3. Select the My Netscape icon on the toolbar.
  4. A warning dialog should appear which states that they connection has failed because the browser is offline. Click OK.
  5. From the File menu, select Offline and then select Work Online. The status bar should display the message "You are online."
  6. Select the My Netscape icon on the toolbar again.
  7. The browser should load the My Netscape page.

Link(s):

Your personal My Netscape page

Expected Results:

Selecting offline should disable the internet connection through the browser and selecting online should restore internet connectivity.
