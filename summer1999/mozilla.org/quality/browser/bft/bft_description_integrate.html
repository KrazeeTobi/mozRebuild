Basic Functional Test Case Description - Application Integration

Initial Conditions:

None.

Description:

To switch between functions listed under the File and Tasks menus.

Accessing Functions from the File Menu

  1. Launch Navigator.
  2. From the File menu, select New Browser Window. A new browser window opens.
  3. From the File menu, select New, then select Mail Message. A new blank email window opens in Messenger.
  4. From the File menu, select New, then select Chat Window. A new chat window opens.
  5. From the File menu, select New, then select Blank Page. A new Editor window opens.
  6. Close all windows except the initial browser window.

Accessing Functions from the Tasks Menu

  1. Launch Navigator.
  2. From the Tasks menu, select Navigator. A new browser window opens.
  3. From the Tasks menu, select Messenger. The Messenger module opens in a window.
  4. From the Tasks menu, select Editor. A new Editor window opens.
  5. Close all windows except the initial browser window.

Link(s):

None.

Expected Results:

Selecting menu items from the File or Tasks menus should open additional or new functional windows.
