WebShell Main Page

By Travis Bogard
Last Modified: 10-7-99


Welcome to the WebShell page.  This page is being written pretty much after the fact and is here simply to provide some docs about WebShell to show a contrast with the new design of the nsWebBrowser and DocShell.  Note that upon completion of Embed and DocShell WebShell's live and use will be abruptly halted.  Anyone having detailed insight on WebShell that would like to provide information on it for historical reasons please let me know so we can update this doc.  But note this doc in general will not be a living document and will probably never be complete as the actual code is on it's way out too.
