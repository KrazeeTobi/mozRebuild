This site is here for you. If it's not helping you, let us know how it can!

For most suggestions about the site, or future directions of the project, or whatever, you might want to just use one of the public mozilla.org mailing lists or newsgroups. We all read those lists, and we strongly recommend that you post your suggestions there.

But, if you would like to suggest something to us privately for whatever reason, you can use the address suggestions@mozilla.org.

If there seems to be some technical problem with the web site itself, please notify webmaster@mozilla.org.

If you are a developer, and you would like to report a bug in the source code distributed by mozilla.org, please use our bug reporting page.

However, if you would like to report a bug in the Netscape Communicator end-user software, you're in the wrong place: head on over to netscape.com instead.
