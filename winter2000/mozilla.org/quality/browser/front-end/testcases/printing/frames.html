Test Case Description for Frames.

Purpose: 
 Determine that Seamonkey prints pages with frames properly 
Initial Conditions: 
 None 
Steps/Description: 
 0. Launch Apprunner
1. Click here to view a page with horizontal frames
2. Click here to view a page with vertical frames.
3. Once each page loads, select menu option "FILE| PRINT
4. Press OK on Print Dialog Box to print the page. 
Expected Results: 
 After Step 4, the page should be printed as seen on screen
