by Lisa Chiang <lchiang@netscape.com>
last modified 08/02/99 13:11

As more of the feature test specifications for each feature area get written, they will be linked off this page. Some of these test specifications are written in "preparation" of testing a feature once it's implemented. Having the test spec here doesn't necessarily mean that the feature being tested is available yet. If you want to know when a feature is to be implemented, take a look at the engineering milestone page. These test specifications are living documents. The tests outlined are manual at this time.

Think any areas are missing from here? If so, let me know. (It'll be even better if you can write the test specs for them!)
