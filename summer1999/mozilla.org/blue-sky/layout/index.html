HTML version 4.0 has been completed. There are those who disagree philosophically with the idea that standards should be specified by the W3C and then implemented by others, as opposed to the IETF model where implementations precede standardization. Regardless, HTML 4.0 is very likely where it's at in the near future, and it's going to need an implementation.
