Basic Functional Test Case Description - Font Size

Initial Conditions:

None.

Description:

To test the Increase/Decrease font size commands in Navigator.

Increase Font Size:

  1. Launch Navigator.
  2. Click on Netscape.
  3. From the View menu, select Increase Font (Increase Font Size).
  4. Verify that the text is larger.
  5. Use the Increase Font (Increase Font Size) keyboard command Ctrl+]. You may need to select the option several times to see a result.
  6. Verify that the text is larger.

Decrease Font Size:

  1. Click on Netscape.
  2. From the View menu, select Decrease Font (Decrease Font Size).
  3. Verify that the text is smaller.
  4. Use the Decrease Font (Decrease Font Size) keyboard command Ctrl+[. You may need to select the option several times to see a result.
  5. Verify that the text is smaller.

Links:

Netscape

Expected Results:

The current page's body text should increase or decrease in size.
