Current Status:
    [May 29, 1999]
  • Autoconf is limping on Rhapsody5.3, we need to take glib/gtk+ out and get libIDL building. See the Unix Build page for more info.
  • The default compiler shipped with Rhapsody 1.0 is gcc-2.7.2.1, Solaris gcc-2.7.2.3 works so we're not that far off the mark. A current version of egcs would be a much stronger compiler.

    • widget/src/rhapsody: This is where the HTML area UI objects
      like buttons and ListBoxes get defined.
    • gfx: Basic drawing primitives need to be written (YB).
  • Get webshell/tests/viewer to build & link
