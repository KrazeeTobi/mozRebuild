Welcome to Mozilla Mail & News
Mozilla Mail & News is a cross platform client with support for SMTP, POP3, IMAP, NNTP, a three pane UI, searching and filtering, address book, and multiple identities. Use Mozilla to access any number of different POP and IMAP mail accounts. This is one of Mozilla's most frequently requested features. Read the release notes for the list of features implemented as of M13.

We Welcome Bug Reports And Feedback
This build has been made available for testing purposes so please help out by trying the software and reporting problems to Bugzilla. The Milestone 12 release notes contains a list of known issues. These are some of the bugs we already know about. Read our Getting Started document for more information on testing Mozilla Mail & News.

Use the newsgroup netscape.public.mozilla.mail-news to contact us.
