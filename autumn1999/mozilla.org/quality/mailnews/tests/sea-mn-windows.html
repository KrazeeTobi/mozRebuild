Feature Test Spec

        5.0 Project
        Mail and News Component
        Windows

Written by:  Ninoschka Baca
History
Date Change    Description    Updated by
3/21/1999        Created         Ninoschka Baca
11/4/1999        Updated        Ninoschka Baca

References:
UI Specification: Netscape Mail UI Specification (currently not available)
Other: Message Compose Specification,  Address Book Specification (currently not available)

  1. Windows to be  tested:
    1. 3-Pane Window
    2. Message Window
    3. HTML  Compose Window
    4. Plain Compose Window
    5. Address Book Window
    6. Print Preview Window
  2. Properties of the Windows are present:
    1. 3-Pane Window
      1. Title Bar
      2. Menu Bar
      3. Mail Toolbar
      4. Side Bar (left side of window)
        1. Mail Panel (top position)
        2. Customize Panel
          1. Item 1
          2. Item2
      5. Thread Pane (right side and top of window)
        1. Multicolumn lists area
        2. Column widget
      6. Message Pane Controller (Hot Click area)
      7. Message Pane (right side and below the Thread)
        1. Scroll bar extending from Envelope panel through Message panel
        2. Envelope Panel
          1. Subject, From, Date, To, Cc, Bcc
          2. Add to Address Book button
          3. Attachments button
        3. Message Panel
      8. Status Bar
        1. Connected or Not Connected
        2. Mail state such as "Document Done"
        3. Unread:# Total:#
        4. Digitally Signed
        5. Encrypted
      9. Taskbar
    2. Message Window
      1. Title Bar
      2. Menu Bar
      3. Message Toolbar
      4. Address Panel
      5. Message Area
      6. Status Bar
      7. Taskbar
    3. HTML Compose Window
      1. Title Bar
      2. Menu Bar
      3. Compose Toolbar
      4. Address Panel
      5. Formatting Toolbar
      6. Message Compose Area
      7. Status Bar
    4. Plain Compose Window
      1. Title Bar
      2. Menu Bar
      3. Compose Toolbar
      4. Address Pane
      5. Message Compose Area
      6. Status Bar
    5. Address Book Window
      1. Title Bar
      2. Menu Bar
      3. Address Book Toolbar
      4. Sorting Columns
      5. Results Pane
      6. Card Pane
      7. Status Bar
      8. SideBar/Address Book Panel
      9. SideBar/ Customize Panel
    6. Print Preview Window
      1. Title Bar
      2. Action Buttons (on bar at top of page)
      3. Status and Progress Bar
      4. Taskbar
  3.  Windows Appearance - visuals:
    1. verify each Window contains the specific properties per Mail and News Specification
    2. verify each Window displays tooltips for each icon
    3. verify the status bar displays text when mousing over a menu item or toolbar
    4. verify that resizing the window works and that all of the properties are still correct (this is also covered in Toolbar test, just do it once)
      1. drag  left to right
      2. drag bottom up
      3. drag bottom left corner to upper right corner
      4. maximize
      5. minimize
    5. verify scroll bars display as required
    6. verify the Netscape logo is displayed on topmost toolbar on the far right (except for Print Preview)
  4. Windows - functionality:
    1. verify Window selected is brought into focus
    2. verify Title Bar displays correct name for Window selected and is in proper format
    3. verify when Window is mimized, correct title is displayed on Windows Start bar (same as Title Bar display only truncated)
    4. Toolbar functionality (covered in Toolbar tests)
    5. Menu Bar functionality (covered in Menu tests)
    6. verify Action buttons perform the correct action (Print Preview)
    7. verify widgets and icons work and are displayed
    8. verify widgets and icons display tooltips describing the function
    9. verify columns can be rearranged, resized and hidden/shown (3-Pane, Thread, Address Book)
    10. verify columns can be sorted (Thread, Address Book)
    11. verify scroll bars scroll correctly and are fully accessible
    12. verify Status Bar displays correct text when mousing over menus and toolbars
    13. verify the progress bar displays progress of an action
    14. verify Type-in filter boxes works correctly (Address Book)
    15. verify drop-down list display correctly (Thread,Compose)
