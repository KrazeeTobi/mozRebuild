XPToolkit
August 13
Submitted by Peter Trudelle <trudelle@netscape.com>

Highlights

  • Resolved 79 bugs in the last week, see our  Bug List  for details.
  • Completed most planned M9 tasks, see recent progress reports and the M9 feature tracking list for details.
Lowlights
  • Several tasks originally planned for M9 slipped, mostly due to unplanned extra time spent assisting the Necko landing, assisting XP developers on box problems, fixing more blocker bugs, and other higher-priority tasks.

    • These tasks have been deferred to M10:
    9615 sched: Convert appcores to components (work done but lost)
     9620 sched: Filter by format Mac: Put filter in dialog (no OS support)
     9621 sched: Filter by format (mac) Specify file format (no OS support)
     9623 sched: Finish splitter
     9625 sched: Fix nsiEventQueue
     9626 sched: window.open mac issues
     9627 sched: window.open linux issues
     9630 sched: Solve the modal dialog thread problem SNAKE (mostly fixed)
     9635 sched: Default buttons in dialogs

    These tasks have been deferred to M11:
     9634 sched: NSPR test apps
     9637 sched: Font Picker Data Source
     9638 sched: Font Picker List box and dropdown box builders for RDF
     9647 sched: Color Picker - Design requirements, implementation meetings
     9648 sched: Color Picker Implementation - XUL syntax
     9649 sched: Color Picker - Hooking up the native event system
     9651 sched: Ensure unsigned XUL can manipulate its own window

    This task has been deferred to M15:
     9624 sched: Embedding APIs
     

  • One other task,  9636 sched: Add GFX scrollbars to Gecko , is substantially completed, but checkin was deferred until M10 in order to allow more time for thorough testing (not really a lowlight).
Accomplishments
  1. Eric Vaughan made good progress integrating GFX scrollbars into Necko. He is tracking down one remaining bug, but we are deferring landing this until M10 in order to allow more time for thorough testing. (evaughan)

  2. Dan Matejka added modal dialog support from JS. (danm)
Priorities
  1. Help xpapps with dialog issues. (evaughan)

  2. Make splitter split. (evaughan)

  3. Add alignment to boxes. (evaughan)

  4. Fix collapse in boxes. (evaughan)

  5. IDL''ify d&d stuff. (pinkerton)

  6. Drag start event. (pinkerton)

  7. Fix d&d/transferable api's once and for all. (pinkerton)

  8.  To complete the basic re-architecture of XP popups, tooltips, and context menus. (hyatt)

  9. Tree widget bugs. (hyatt)

  10. Tree widget scheduled tasks. (hyatt)

  11. Get dialogs working in all situations. They are especially squirrelly on gtk, and when called from certain places in the code, as reported in my M10 bugs.  (danm)

  12. Keybinding bug fixes. (saari)

  13. XPMenus ahoy. (saari)
