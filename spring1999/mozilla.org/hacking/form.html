The CVS Contributor Form

If you want to get write access to the CVS tree, you'll need to fill out the CVS Contributor Form. This is PostScript (yes, we know) and apparently ghostview chokes on it (yes, we know), but it's all we've got. Print it, mourn the loss of a tree, fill it out, sign it, fax it to the attention of Leaf Nunes at 650.937.5413 and then send the signed original via postal mail to:

Leaf Nunes
Netscape Communications Corp.
Mailstop MV-094
501 E. Middlefield Rd.
Mountain View, California 94043
USA
You will need someone to ``vouch'' for you as well. This person should be reasonably experienced with checking code in to the Mozilla source tree. If you have trouble finding a voucher, we can help you. If all of that is taken care of, we'll be happy to give you an account on the CVS server -- mail leaf@mozilla.org. In the meantime, read our hacking guidelines -- you'll be tested on this material!
