Basic Functional Test Case Description - Send Page

Initial Conditions:

None.

Description:

To use the Send Page feature from Navigator.

Using Send Page from the Edit menu

  1. Launch Navigator.
  2. Click on this Netscape link.
  3. From the File menu, select Send Page. The url "http://home.netscape.com" should be inserted into the text entry area of the message window, and the subject field should be labeled "Netcenter".
  4. Type in your email address as the recipient and click on the Send icon.
  5. In Messenger, verify that the new message received includes the inserted Netscape page.

Link(s):

Netscape

Expected Results:

The Netscape page should be inserted into the text body.
