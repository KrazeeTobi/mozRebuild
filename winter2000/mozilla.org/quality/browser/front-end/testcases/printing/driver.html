  How do I obtain printer driver information?

 On Windows, select menu option Start |Settings|Printers from the task bar. A window titled Printers opens up. Right click on a printer and select Properties from the popup menu. Click on the Print Test Page button. This test page gives details for the printer driver.

 On Mac OS, select the Apple System Profiler option from the Apple menu. Click on the System Profile tab. Expand the Printer Overview option to view the printer listing.  In the pane titled Desktop Printer(default), the printer driver version can be seen under the heading Printer version and the printer driver name can be seen under the heading Printer driver.

On a Linux window, please type the following command.  " rpm -qa | grep lpr " . The printer driver version gets printed out on the console. .
