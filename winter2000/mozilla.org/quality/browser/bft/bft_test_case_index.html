Basic Functional Test Case Index - Browser

The test cases listed in the left frame are to be used to perform a quick browser verification test. When a test case is selected, the test case will be displayed in a new browser window. The test cases are organized into two major categories: Function or HTML element / CSS. The functional test cases are to be used to test certain browser functions. The HTML element / CSS test cases are to be used to test how the browser renders and supports certain elements and properties.

The test cases are also organized into two types of tests: functional steps and visual samples. The visual sample tests have an associated description file. The description file, if selected, will be displayed within this frame. The functional step and visual test files will be displayed in a new browser window.
