getMIMEDescription

public java.lang.String getMIMEDescription()
Returns the MIME description for the pluglet. The MIME description is a colon-separated string containg the pluglet MIME type, pluglet data file extension, and pluglet name, for example: "application/x-simple-pluglet:smp:Sample Pluglet"
