Basic Functional Test Case Description - FTP

Initial Conditions:

Description:

To test the FTP protocol using Navigator's Upload File command.

  1. Launch Navigator.
  2. Select ftp file.
  3. From the File menu, select Save As. Enter a new file name and click the Save button.
  4. NOTE: For this step, you will need access to a server that you have read/write access to. In the URL location field, type "ftp://enter the serer location here" and press the return/enter key.
  5. From the File menu, select Upload File.
  6. In the File Upload dialog box, select the new file and click on the Open button.
  7. After the file is copied, open the file and verify that the file is displayed correctly.

Links:

ftp file

Expected Results:

The local file is copied into the selected directory on the selected server.
