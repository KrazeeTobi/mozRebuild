Class grendel.storage.NewsRCUpdateThread 

java.lang.Object
   |
   +----java.lang.Thread
           |
           +----grendel.storage.NewsRCUpdateThread

This class implements a thread which runs in the background and saves a modified newsrc file. It dies once the file has been updated; it will be re-spawned if that newsrc file becomes dirty again.

There can be one of these threads per NewsRC object.
