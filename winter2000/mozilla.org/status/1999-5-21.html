Editor
May 21
Submitted by Akkana Peck <akkana@netscape.com>

Akkana Peck has our Editor update::

M6 bug fixes. It's really starting to look like an editor in M6! We have open file, edit page, new blank page, save, and save as hooked up now -- so you can use the editor to do editing. Other things recently fixed or improved: the find dialog, arrow keys for moving around, some selection improvements (sometimes you can select images as well as text), more text editing rules, and more dialog improvements.
