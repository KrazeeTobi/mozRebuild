XPToolkit Channel Summary

For documenation, specs, requirements, and general project stuff, take a gander at the XPToolkit home page

Tons and Tons O' New Documentation

hyatt, in a mad fit of desperation, has rewritten the bulk of our documentation from scratch. Go Dave!! In addition, Pete Collins has been writing a XUL User's Manual for us. Thanks Pete!

Mac/Win32 Drag And Drop Operational

Not perfect by any means, but you can now drag plain text from outside apps into Ender on MacOS and Win32. Next up is being able to start drags from within Mozilla itself.

Context Menus Operational

Popup menus and context menus are pretty close to being finished. Context menus aren't perfect on mac and because of unimplemented APIs in the DOM (discovered at the last minute), menus only appear at 50, 50. Don't worry, we're working our butts off to get this all working!

scc Reports On Code Bloat

scc reports on code generation size tests comparing raw pointers, nsCOMPtrs and nsIPtr's. The net result? nsCOMPtr is good.

window.openDialog() Operational

danm tells me that window.open() can open dialogs and fill in values with parameters. Modal dialogs aren't working yet, but that's for other reasons. It's documented at Dialog Cookbook and XUL Language.
