Writing Skinnable XUL & CSS

David Hyatt just wrote a must read document on what to do and what not to do when writing your own skin. Everyone should look at this if they are interested in skinning the beast.

Clipboad & Drag&Drop docs online

Three new documents have just been written:

  • Intro to Clipboard/Drag & Drop
  • The Clipboard
  • Drag and Drop
  • Added D&D info to toolbar docs.
  • Added D&D info to tree docs.

New splitter docs online

Docs on the splitter widget have been added to the xptoolkit homepage.

Visual debugging box docs online

Newly added to the existing box documentation is a blurb at the bottom on how to visually debug boxes. Very very helpful. Check it out.

Hyatt just spanked the tree with changes to make xul.css load from ua.css so you no longer need to include it in your xul files.
