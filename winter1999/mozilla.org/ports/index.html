ports

Many different groups are planning ports of Mozilla to new platforms and toolkits. This page will contain a list of all known-active ports of Mozilla, as well as how to find out more about them.

As with other Mozilla-derived projects, if you are organizing a serious porting effort of some kind, and would like us to host the web pages and mailing lists about that project here on mozilla.org, let's talk.

Pages for the following porting efforts are currently hosted here. Note that these pages are maintained by others, not by the staff of mozilla.org.

  • Rhapsody, a port to Apple's next-generation operating system.

  • GTK, a port of the Unix front-end to an alternative toolkit to Motif.

  • OS/2, a port to the IBM's OS/2 operating system.

Off-site links:

  • BeOS, a port to the Be operating system.

  • Troll Tech ported the Unix front-end to their product, the Qt toolkit (another Motif alternative.)

  • Some folks are interested in rewriting Mozilla from scratch in Java. I (jwz) think this is a bad idea for many reasons, but if you are not deterrable, you can find their web pages here.

  • Far more interesting on the Java front is Steve Wynne's page investigating the task of plugging a free Java implementation back in to Mozilla.

  • aMozilla, a port for the AmigaOS (PPC/68k).

Do you know of other porting efforts? Let us know!
