    Ensuring the localizability - XUL Coding Style Guidelines

    UTF-8 encoding. Wherever applicable, the recommended encoding of the content data and attribute values is UTF-8. In Mozilla, when such information is not specified, UTF-8 is the default encoding of all XML, XUL, and RDF documents.

    Chrome URLs. All resource references in XUL shall use the chrome type URLs instead of hardwired URLs.
