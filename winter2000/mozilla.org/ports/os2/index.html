warpzilla

the mozilla port to os/2

Download latest Warpzilla

News!

Warpzilla is a project to bring the leading-edge browser to the OS/2 desktop. This homepage is designed to deliver information both to end users and to developers who want to be informed about the project or who even want to get involved in it. Everyone is invited to participate (in almost any way they want) because there are still lots and lots of things to do.

To get involved, here is the scenario: you first obtain the latest code level, then make some useful modifications (the hard part) and tell everyone about your patches. Eventually your changes will be checked back in to the code repository so that they become accessible to everyone.

For the coders, here are some pointers to get you started:

Peek at our latest assignment page.

You don't know anything about programming? Don't despair! The project also needs help on graphics and documentation (these are mostly cross-platform endeavors though). Try prowling around www.mozilla.org on the left side of your screen looking for a suitable project and get to it!
We would cheerfully welcome someone donating a machine to run a constantly updating tinderbox.

For the bold and daring who want to be testers, some recent binaries are floating around.
