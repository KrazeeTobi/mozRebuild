Things you can help us develop-

So you wanna write code for Netlib? Well, here is your chance. Here is a list of what you as a developer, could help us with. If you think any of these items match your interest, go ahead, knock yourself out. But before you do that, send me a mail so that I know you are working on it. I will also keep updating this page with who's working on what.

HTTP/1.1 Digest Access Authentication See RFC 2069 for details. Matthew Marlowe, Chris Austin-Lane. Contact us.
