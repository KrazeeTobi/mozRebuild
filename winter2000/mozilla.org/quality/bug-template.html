The following template is the suggested format to cut/paste into the Description area of a Bugzilla New Bug.  Once you have pasted this into a new bug, click on the "Remember values as a bookmarkable template" button to set yourself up for bookmark access to bug writing.  NOTE: You will need a bookmark for each product - Browser, Mail, etc.

For full information on fields in Bugzilla and how to use please see:
http://www.mozilla.org/quality/bug-writing-guidelines.html

Please follow this format to ensure a fast turnaround on your bugs!
