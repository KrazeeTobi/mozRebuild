help qualify m14

We need you to help test Mozilla, and let us know whether you think we have a Milestone quality build.

We need people who can test different parts of the browser, on as many platforms as possible. Here are some suggested starting points:

  • Browser/Editor Tests
  • MailNews Tests
  • Smoke Tests
  • Layout Tests
  • David Baron's CSS Tests
  • David Baron's DOM Tests
  • Tests under Mozilla's Debug Menu
If you run any of these test suites (or even any part of these suites), please post to netscape.public.qa.general or send mail to cbegle@mozilla.org, and let us know:
  • What platform you tested on
  • What tests you ran
Even if you can't help run these tests, you can still help out:
  • Download one of the nightly builds, and use it as your main browser, editor, and mailer.

    If you're on a 32-bit Windows platform, download the candidate builds that have Full Circle, and end us data whenever you crash.

  • Complete the survey at www.mozillazine.org, and let us know what you think of this buid.

  • Report bugs on the M14 candidate builds. Please add the keyword M14 to the keywords field.
