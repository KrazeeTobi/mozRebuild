Basic Functional Test Case Description - CSS 1 (#2)

Requirements:

- bft_browser_css2.html
- upper.css
- tn_mblew.jpg
- k9.gif
- aladdin.gif
- catrun.gif

Description:

The purpose of this test is to verify that the text and image objects are placedand displayed correctly via CSS commands. Each text string and image is set to a particularfont, size and location.

  1. Launch Navigator
  2. Display the CSS #2 test case.

Link(s):

CSS sample 2

Expected Results:

Background:Should be antique white, with an image displayed on far right of K9 (k9.gif).
H1:The level 1 header text should be blue with a thin, red, grooved border.
Aladdin lamp:(aladdin.gif) - Flush left.
Marvin:(tn_mblew.jpg) - Flush with the right side of the aladdin lamp.
Nested tables:There should be three visible tables nested within each other.
Base table:Consists of three columns and one row. The first column contains text, the second column contains an animated gif, and the third column contains a nested table,
First nested tableConsists of three rows and one column. The first row contains text, the second row contains an animated gif, and the third row contains a nested table.
Second nested tableConsists of two columns and one row. The first column contains text and the second column contains an animated gif.
