Feature Test Spec

5.0 Project (seamonkey)
Mail and News Component
Layout (Display of Messages)

Written by: Peter Mock

History:
121598        Created
010499        Updated

References:
UI Specification:   Composition UI spec
                            Master Messenger UI


Description: These tests applies to Mail and News.

Compatibility displaying messages from within 5.0 itself.
Purpose: The message contents should display the same in the Message Pane and Message Window.

  1. Display the same mail message in the message pane and message window.
    1. Verify the message contents are the same in both displays.
    2. Verify you should not be able to open more than one message window of the same message
    3. Verify you should not be able to open more than one messenger window displaying the same messages
    4. Verify if you file this message, does the window or pane close (tbd)
    5. Verify if you delete this message, does the window or pane close (tbd)
  2. View messages with different contents
    1. Test messages to be created.


Compatibility displaying messages from other mail programs
Purpose: 1) To test displaying messages sent from other Communicator platforms and versions.  2) To test displaying messages sent from 3rd party vendors.  Note: the number of messages to be tested for from each program will be determine by the available time.  For each test, start by sending a mail message from seamonkey.

  1. View messages sent from seamonkey. Verify the messages are displayed correctly.
    1. Windows release
    2. Mac release
    3. Unix release
  2. View messages sent from previous versions of Communicator. Verify the messages are displayed correctly.
    1. Communicator 4.7
    2. Communicator 4.08
    3. Navigator 3.04
  3. View messages sent from current Microsoft Applications. Verify the messages are displayed correctly.
    1. Outlook Express 5 (PC)
    2. Outlook Express 4.5 (Mac)
    3. Outlook 98
    4. Outlook 2000
    5. Exchange
  4. View messages sent from Eudora. Verify the messages are displayed correctly. Verify the messages are displayed correctly.
    1. Eudora for PC
    2. Eudora for Mac
    3. Eudora Lite (free downloadable from eudora web site)
  5. View messages sent from unix base programs included with popular unix operating systems. Verify the messages are displayed correctly.
    1. Linux (RedHat)
    2. Solaris
    3. TBD
  6. View messages sent from other mail popular mail programs such as:
    1. Emailer-Lite included with earlier versions of Mac OS.
    2. (TBD)

Security Test
(In Communicator 4.x, it was susceptible to buffer over problem that we wish to test for.  Use test messages from 4.x generated from engineering)

  1. View mail messages that has a subject area which caused a buffer overrun in 4.5
  2. View mail messages that has a message area which caused a buffer overrun in 4.5
  3. View mail messages that contains one or more attachments which caused a buffer overrun in 4.5
