Basic Functional Test Case Description - Preferences

Initial Conditions:

None.

Description:

To test the basic functionality of the Browser's Preference dialog.

  1. Launch Navigator.
  2. From the Edit menu, select Preferences.
  3. Verify that the following categories appear in the Preferences dialog:
    • Appearance
    • Navigator
    • Mail & Newsgroups
    • Composer
    • Roaming Access
    • Offline
    • Advanced
  4. Select the Navigator category. The Navigator preferences dialog will be displayed.
  5. In the Home Page Location text entry field, type in http://www.mozilla.org.
  6. In the Navigator starts with section, select the radio button associated with Blank page.
  7. Select the Smart Browsing sub-category and uncheck "What's Related."
  8. Select the Advanced category and uncheck "Automatically load images."
  9. Click OK.
  10. Quit and re-launch Navigator. The browser window should be blank.
  11. Select the [HOME] icon on the toolbar. The Mozilla org home page should display without the images, and the [What's Related] icon should not appear on the toolbar.

Link(s):

None.

Expected Results:

Based on the changes you made in the Preference dialog, Navigator should open a blank document as its default window. Selecting the "Home" toolbar icon should display the new home page url. After opening the new home page, images on the page should not be displayed. Finally, the "What's Related" button will not be displayed.
