Welcome to the
Warpzilla Homepage

This is the official homepage of the Warpzilla project. In short, the Warpzilla project has been founded to port the now freely available source code of the Netscape Navigator (aka Mozilla) to the OS/2 Warp platform.
This homepage is designed to deliver information both to end users and to developers who want to be informed about the project or who even want to get involved in.

Please select one of the links on the left side for more information.
