 How to Create Address Books, Cards, Lists, Directories, etc. (n/a for Seamonkey at this time)

Address Book

  1. Launch Communicator
  2. Go to File menu Communicator|Address Book
  3. Go to File menu New Address Book
  4. Type in a name for your New Address Book, click OK
  5. The New Address Book should be listed in the left pane of the Address Book
New Card
  1. Launch Communicator
  2. Go to File menu Communicator|Address Book
  3. Select the target Address Book (where you want the Card to be located)
  4. Go to File menu New Card or click on the New Card toolbar button
  5. Verify the selected address book is the target address book in the pull-down list at the top of the New Card window.
  6. Fill in the First Name, Last Name (Display Name will automatically fill when entering FN & LN) for your New Card, fill in other fields as needed for testing specific scenarios. Click OK
  7. The New Card should be listed in the right pane of the target Address Book
New List
  1. Launch Communicator
  2. Go to File menu Communicator|Address Book
  3. Select the target Address Book (where you want the List to be located)
  4. Go to File menu New List or click on the New List toolbar button
  5. Verify the selected address book is the target address book in the pull-down list at the top of the New List window.
  6. Type in a name for your New List, Nickname (optional), Description (optional) and cards (see below), then click OK
  7. Add cards using:
    • Drag n Drop a name from another address book
    • Type a name in the list area of the New List window
    • Drag n Drop a name from a header in an email message
  8. The New List should be listed in the left pane under the target Address Book, AND in the right pane if the target Address Book is selected.
  9. Any cards that were not in the target address book will also be added to the address book at the top level.
New Directory
  1. Launch Communicator
  2. Go to File menu Communicator|Address Book
  3. Go to File menu New Directory
  4. Fill in a information for your New Directory (must give a Description and LDAP server). Examples of other directories listed below:
    • Infospace- LDAP server is ldap.infospace.com, other information is not necessary, Port will automatically be entered.
    • Bigfoot- LDAP server is ldap.bigfoot.com, other information is not necessary, Port will automatically be entered.
    • WhoWhere- LDAP server is ldap.whowhere.com, other information is not necessary, Port will automatically be entered.
  5. The New Directory should be listed in the left pane

Adding the Netscape Phone Book to your Directory list through the Navigator window
  1. Launch Communicator
  2. Go to File menu Communicator|Navigator
  3. Enter into the Location field http://phonebook.mcom.com then press the Enter key
  4. Use the vertical scroll bar to move to the bottom of the page
  5. Click on the Import Phonebook link
    • In the AddressBook, the following information was added.
      • Description: Netscape Phonebook
      • LDAP Server: directory.netscape.com
      • Search Root: o=Netscape Communications Corp., c=us
      • Port: 389
Replicating Directory(s)
Method 1
  1. Launch Communicator
  2. Go to File menu Communicator|Address Book
  3. Select the directory in the left pane that you wish to replicate
  4. Go to Edit|Properties or click on the Properties toolbar button
  5. Click on the Offline Setting tab to bring it into the foreground
  6. Check the option “Select this directory for download”
  7. Click on the Update Now button to start the replication process
  8. The replicating directory progress bar should appear
Method 2
  1. Launch Communicator
  2. Go to File menu Communicator|Messenger
  3. Go to File menu File|Offline-> Synchronize
  4. Uncheck the option to synchronize Mail Messages
  5. Uncheck the option to synchronize Newsgroup Messages
  6. Check the option to synchronize Directory Entries
  7. Click on the Select Items button
  8. Use the vertical scroll bar to scroll to the directories listing after the mail folders and subscribed newsgroups.
  9. Expand the collapsed Directory list to show the directories that are available for replication
  10. Beneath the Download column, check the desired directory(s) that you wish to replicate
  11. Click on the OK button to return to the Synchronize Offline Items dialog
  12. Click on the Synchronize button to start the replication process
