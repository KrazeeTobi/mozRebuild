Basic Functional Test Description - Imagemap

Requirements:

- bft_browser_imagemap.html
- imgmap.gif
- bft_browser_html_mix2.html

Description:

This test verifies that Navigator correctly displays an image with a selectable area.

  1. Launch Navigator.
  2. Display the IMAGEMAP element test case.
  3. Mouse over the dog bones and paw prints. The cursor should remain as a pointer.
  4. Mouse over the dog face. The cursor should appear as the pointing hand.
  5. Select the imagemap area.

Link(s):

Imagemap test page

Expected Results:

The page should display without error. The image should be displayed correctly, and the imagemap area should be functional.
