Feature Test Spec

        5.0 Project
        Mail and News Component
        Toolbars

Written by:  Ninoschka Baca
History
Date Change    Description    Updated by
3/29/1999        Created         Ninoschka Baca
11/4/1999        Updated        Ninoschka Baca

References:
UI Specification:  Netscape Mail UI Specification (currently not available)
Other:  Message Compose Specification, Address Book Specification (currently not available)

  1. Windows displaying the Toolbar for testing:
    1. 3-Pane Window
    2. Message Window
    3. HTML  Compose Window
    4. Plain Compose Window
    5. Address Book Window
  2. Toolbars to test:
    1. Application Toolbar (3-pane, Message Window)
    2. Taskbar (3-pane, Message Window, HTML Compose Window, Plain Compose Window, Address Book Window)
    3. Formatting Toolbar (HTML Compose Window)
    4. Address Book Toolbar
  3.  Toolbar  Appearance - visual:
    1. verify the appearance and order of each toolbar button for each Special folder
      1. Inbox
      2. Outbox/Unsent Messages
      3. Drafts
      4. Sent
      5. Trash
      6. News
    2. verify that each button, when clicked, depresses
    3. verify that each button, when mousing over, displays as a button
    4. verify the tooltips state the correct action for each button and matches the specification
    5. verify the Toolbar looks correct in different video resolutions and color palettes for your video card
      1. 800x600  resolution
      2. 1024x800 resolution
      3. High Color (16bit) color palette
      4. 256 color palette
      5. True Color (24bit) color palette
      6. 256 Colors (mac) color palette
      7. 256 greys (mac) color palette
      8. million's of colors (mac) color palette
      9. thousands of color (mac) color palette
    6. verify the Toolbar looks correct after resizing
    7. verify that Collapsing/Expanding the toolbars works
  4. Toolbar Button - functionality
    1. verify that each button, when clicked, brings up the proper dialog or performs the correct action
    2. verify that you can close the dialog brought up by the button action
    3. verify that for toolbar buttons with a down arrow, a single click performs the action of the first menu item on the drop-down list, click and hold drops down a list to select from.
