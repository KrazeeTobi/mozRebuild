Feature Test Spec

5.0 Project (Seamonkey)
Mail and News Component
Basic Mail and New Functionality: Print

Original document was written by: Peter Mock for Nova
Modified and maintained by: Fenella Gor for 5.0 (Seamonkey)

References:

5.0 Seamonkey UI Specification:
Test case management system: n/a
Approx. number of test cases: 260

Description: Printing involves out-putting the displayed information on-to the printed media or electronic file. This test plan focuses on printing the content of the mail message to a device/file.Verify the expected results visually after each test and that all elements of the email messages has printed out correctly.

  1. Mail can be printed from various Mail/News windows
    1. 3 Pane UI
    2. 2 Pane UI (3 Pane UI collapsed to 2 pane UI)
      1. Collapse the Thread pane only and print a message
      2. Collapse the Folder pane only and print a message
    3. 1 Pane UI (3 Pane UI collapsed to 1 pane UI)
      1. Collapse the Thread pane and Folder pane then print a message
    4. Message Window
    5. Compose Window (if possible)

  2. Ways to Print a Message
    1. Menu (File|Print)
    2. Toolbar (Print icon)
    3. Mnemonics (CTRL P)
    4. Context Menu

  3. Print Preview
    1. View Next page
    2. View Previous Page
    3. View Two Pages
    4. Zoom In
    5. Zoom Out
    6. Print
    7. Close/Cancel print preview

  4. Print Setup ( Selected from Print Properties)

    2. Margins
      1. Top
      2. Bottom
      3. Left
      4. Right
    3. Headers
      1. Document Title
      2. Document Location (URL)
    4. Footers
      1. Page Number
      2. Page Total
      3. Date Printing
    5. Printer specific options
      1. Duplex
      2. Postscript
      3. Scaling
      4. 1 UP/ 2UP/ 4 UP

  5. Number of messages that can be Printed
    1. Single message
    2. Multiple messages (Request for Enhancement)
    3. Selective messages (Request for Enhancement)
    4. All messages (Request for Enhancement)

  6. Mail message and news articles with the following contents:

    7. Background Image/texture
    8. Other
