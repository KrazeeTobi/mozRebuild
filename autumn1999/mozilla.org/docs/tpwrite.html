contribute documentation

There are many documentation issues for mozilla.org that are yet to be worked out, including copyright policies and submission procedures.

Copyrights

For now, we are using Netscape's standard copyright on the documents posted here. Other authors who submit documents to be posted on this site will retain their own copyright.

In the future, we hope to institute a copyright policy that encourages information sharing, possibly like the Linux Documentation Project in which authors retain their own copyright while extending blanket reproduction and distribution privileges (subject to certain conditions). We are interested to learn your opinions about this issue while we work it out.

Submissions

Our temporary procedure for submitting a document is to send a mail attachment, preferably in HTML, to suggestions@mozilla.org.

Eventually, we will have a more formal review procedure to ensure technical accuracy and editorial consistency. For now, we'll figure it out case-by-case. We have been impressed with the expertise in evidence in postings to the mozilla.org discussion groups, and we hope everyone can benefit from that knowledge as seamlessly as possible.

For documentation discussion, visit the mozilla.org documentation newsgroup.
