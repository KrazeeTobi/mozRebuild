Netscape Messaging Access SDK (C/Java): Source Code Release
<prasad@netscape.com>
Last Updated: August 25, 1998

Netscape Messaging Access SDK comprises of a set of APIs that enable you to write applications to access to the services of Messaging Servers (such as Netscape Messaging Server) that conform to Internet Messaging standards.

These APIs are defined in both C and Java languages and are supported on several platforms including Windows NT and Unix.

To promote development of different classes of messaging applications ranging from simple mail-enabled or mail-aware applications to sophisticated messaging access applications these APIs are offered at two different levels:

The Protocol APIs are based on standard Internet messaging protocols: POP3, IMAP4, SMTP, and MIME. They reflect the underlying protocols pretty closely, providing the necessary foundation needed by any higher level functionality.

The Convenience API (Java only) combines the functionality of the SMTP and MIME APIs into a simple but sophisticated message compose, MIME encode and send API to build and send messages with or without attachements. The convenience API is sometimes referred to as the high level API.

The SDK also includes multi-platform low level I/O interfaces and additionally on the C side, the SDK includes a streaming API. The low level API is not documented but it is pretty straight forward. The streaming API is fully documented.

This source code release of the SDK includes the source code for all the APIs on both Java and C sides. The source code for the Netscape Messaging Access SDK is covered by the Netscape Public License.

If you are interested in contributing to the source code for the Netscape Messaging Access SDK, please read the general information provided under "getting involved".

Before you work with the source code, you should read the source code documentation. You can also find information about the Netscape Messaging SDK in the Messaging Access SDK Guide, Java Version and the Messaging Access SDK Guide, C Version.

You can download the source code here. The downloaded file will be in a gzipped gtar format. You need to uncompress and extract the sources from this file using gunzip and tar. You can obtain the gnu versions of these executables from one of the following download sites:

Cygnus (http://www.cygnus.com/misc/gnu-win32)
GNU (http://www.gnu.org/order/ftp.html)
MIT (ftp://prep.ai.mit.edu/pub/gnu)
The extracted source files include CVS information, which means that you can get the latest version of the source through CVS without having to check out the source code explicitly.

To build the SDK, follow the instructions here.

If you have questions or want to discuss issues about the Messaging Access SDK source release, you can post a message to the netscape.public.mozilla.mail-news newsgroup. (For more information about the newsgroups and mailing lists available from mozilla.org, see the community page.)
