Basic Functional Test Case Description - Help

Initial Conditions:

None.

Description:

To open and close the NetHelp dialog window.

Launching NetHelp from the Help menu

  1. Launch Navigator.
  2. From the Help menu, select Help Contents (Help). The NetHelp window should display the "Browsing the Web" section.
  3. Close the window using the X (exit) icon. A dialog box containing the text "Close Window Communicator Help?" should appear.
  4. Click OK.

Launching NetHelp from Preferences

  1. Launch Navigator.
  2. From the Edit menu, select Preferences. This displays the Preferences dialog box.
  3. Select the Navigator category.
  4. Click on the Help button in the Preferences dialog. The NetHelp window should display the "Specifying the Starting Page" section.
  5. Close the window using the X (exit) icon. A dialog box containing the text "Close Window Communicator Help?" should appear.
  6. Click OK.

Links:

None.

Expected Results:

Selecting the Help Contents (Help) from the Help Menu should open the NetHelp window. Clicking on the X (exit) icon on the window should generate a javascript dialog box.
