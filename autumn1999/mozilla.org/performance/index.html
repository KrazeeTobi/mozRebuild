Open Performance Bugs. A list of all the open performance-related bugs in bugzilla, ordered by owner. Think you've found a performance problem? Check here first. If you file a new performance bug, please make sure to put the letters [PERF] in the bug's status whiteboard field.

Mail-News Performance Statistics. A regularly-updated list of mail-news performance stats.

Tools. For measuring performance. Tip o' the propeller-cap to Daniel Roberts (zuperdee@penguinpowered.com) for the pointers.

Profiling JuJu. Things you should know about doing profiling. Tips and tricks for some of the tools, and lots of other Good Things To Know.

References. A dumping ground for performance-related documents.

Community. netscape.public.mozilla.performance is the Mozilla performance newsgroup, and mozilla-performance@mozilla.org is an e-mail mirror.
