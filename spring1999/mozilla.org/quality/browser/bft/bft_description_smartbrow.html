Basic Functional Test Case Description - Smart Browsing

Initial Conditions:

Enable Internet Key Words and What's Related in the Smart Browsing Section of Preferences.

Description:

To test the Internet Key Words and What's Related components of smart browsing.

Using Internet Keywords

  1. Launch Navigator
  2. On the Location Toolbar, within the Location field, type "Los Angeles Times" and press the enter/return key.
  3. The url address should automatically change to "www.latimes.com," and The LA times website should open in the browser window.

Using What's Related

  1. With the LA Times site displayed, click and hold the What's Related button. A listing of related websites will appear in a popup menu.
  2. Select one of the websites listed. The website you selected should open in the browser window.

Link(s):

None.

Expected Results:

Entering the keywords "Los Angeles Times" should automatically send you to the "latimes.com" web site. At this site, clicking on the "What's Related" button should generate a list of related sites. Selecting a site from this list will open it in the browser window.
