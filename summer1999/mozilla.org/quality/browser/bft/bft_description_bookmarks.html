Basic Functional Test Case Description - Bookmarks

Initial Conditions:

None.

Description:

To verify that you can add, edit, and delete bookmarks from Navigator.

  1. Launch Navigator and go to "www.cnn.com".
  2. Use the Bookmark option to add the CNN page to your Bookmark file.
  3. Verify that the bookmark "CNN Interactive" has been added at the bottom of the Bookmark menu.
  4. On the Windows platform, select Edit Bookmarks from the Bookmark popup. On the Macintosh platform, select Edit Bookmarks from the Bookmark menu.
  5. In the Bookmark window, select the CNN bookmark.
  6. Copy and paste the bookmark to make another copy.
  7. On the Windows platform, select the new duplicated bookmark and choose Bookmark Properties from the Edit menu. On the Macintosh platform, select the new duplicated bookmark and choose Get Info from the Edit menu.
  8. In the Name field, type a new name like "CNN's Page." Click OK.
  9. Select the "CNN Interactive" bookmark.
  10. From the Edit menu, select Delete (Windows) or Clear (Macintosh). Close the window.
  11. On the Windows platform, click on the Bookmark icon. On the Macintosh platform, select the Bookmarks menu. Verify that "CNN's Page" is listed.

Link(s):

"www.cnn.com"

Expected Results:

The bookmark should be added, edited, and deleted from Navigator.
