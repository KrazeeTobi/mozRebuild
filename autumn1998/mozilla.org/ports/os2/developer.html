Developer Information

Warpzilla is a project open to everyone. To avoid chaos, the Mozilla group of Netscape has defined a very sophisticated way of development. All official code of the Mozilla project (which Warpzilla is an official part of) is located on the servers of www.mozilla.org. To get involved, you first need to obtain the latest code level from there and then someone of the Mozilla team needs to check back your changes to the code repository so that it becomes an official part of Warpzilla.
