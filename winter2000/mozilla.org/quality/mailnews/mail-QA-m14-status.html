Notes:

  • Definition of PASS/FAIL:
    • PASS rating if the feature set is testable by QA, regardless if it has a bug or two.   There may be days when there is a bad build :-)
    • FAIL rating will be given if a feature is supposed to be implemented for this milestone and it's not in there yet so we cannot perform basic testing on it; or the feature is implemented, but not functional at its very simplest task
