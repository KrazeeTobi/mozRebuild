Basic Functional Test Case Description - JavaScript Support

Initial Conditions:

Enable JavaScript within Preferences. You can find the JavaScript option in the Advanced section of Preferences.

Description:

Open the HTML file that contains the javascript functions.

  1. Launch Navigator
  2. Display the JavaScript test case
  3. Set your computer system clock between 4am and 5pm. Reload the test case, the document that is generated contains a white background and the day image. The alert text wiill state: Good Morning and Afternoon.
  4. Set your computer system clock between 6pm and 3am, the document that is generated contains a black background and the night graphic. The alert text wll state: Good Evening and Good Night!
  5. Set your computer system clock to the correct time

Link(s):

JavaScript test case

Expected Results:

The page should be displayed with the correct background color, with the correct image, and the correct alert dialog with the appropriate text string. If the system clock is between 4am and 5pm the document will display in the day mode. If the system clock is between 6pm and 3am the document will display in the night mode.
