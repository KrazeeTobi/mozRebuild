Test Case Description for highlight-js-text.

Purpose:

Determine whether text output from a JavaScript applet is properly selected in conjunction with other page content.

Initial Conditions:

None.

Steps/Description:

1. View the attached test case.
2. Drag-select from "Browser Javascript Test" to the bottom of the document.
3. Try every ad hoc selection you can think of, to determine whether the JavaScript output is selected differently or inconsistently.

(e.g. selecting from white space to JavaScript & vice-versa, drag-scrolling, shift-selecting, Command/Control selecting, selecting and observing changes in scrolling behavior, etc.)

Expected Results:

After step #2, the entire content of the page should be selected.

After step #3, no problems should be encountered, or selection behaviors that are inferior to those of IE or Navigator 4.x.

Link(s):

highlight-js-text
