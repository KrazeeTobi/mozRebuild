There is a CVS server set up to coordinate development. You can check the gtkfe module out of CVSROOT=:pserver:anonymous@cvs.gnome.org:/debian/home/gnomecvs (password is blank). For help on using CVS, the CVS Bubbles link page is a good start. You will also want to check the gtk+ (and optionally the imlib and gnome-libs modules) out of CVSROOT=:pserver:anonymous@cvs.gnome.org:/debian/home/gnomecvs (password is blank - press <Enter>).
The mozilla-gtk mailing list is the discussion center.
