Basic Functional Test Case Description - Proxies

Initial Conditions:

Clear the disk and memory cache before you begin.

Description:

To navigate to different sites using a proxy server.

Using Proxy Configuration in Navigator

  1. Launch Navigator.
  2. From the Edit menu, select Preferences. The Preferences dialog box will be displayed.
  3. Select the Advanced category, then select the Proxies sub-category.
  4. Select the radio button associated with Manual Proxy Configuration. The View button will be enabled.
  5. Click on the View or Configure button. The Manual Proxy Configuration dialog box will be displayed.
  6. In the HTTP proxy field, type "chainsaw"
  7. In the Port field, type "8056", click OK.
  8. In the Preference dialog, click OK again.
  9. In the URL location field, type "http://www.abcnews.com/".
  10. After this page loads, type in "http://www.universalstudios.com/main.html"

Link(s):

ABC News

Universal Studios

Expected Results:

After setting the proxy connection to "Chainsaw" 8056 (56k simulator), the url sites "abcnews" and "universalstudios" should open and display in the browser.
