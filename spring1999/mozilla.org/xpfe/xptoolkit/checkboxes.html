Checkbox (Tri-State)

[Table of Contents] [Previous: Progress Meter] [Next: The Box System]

Feature Owner: Mike Pinkerton

The XUL checkbox is similar to its HTML counterpart with one notable exception. It is capable of being placed into a "tri-state" mode, and in this mode it has three states: unchecked, checked, and mixed.

The checkbox is declared using the checkbox tag. The value attribute can be set to change the checkbox's state. Possible values are 0, 1 and 2.

Until the value is set to 2, the checkbox behaves just like its HTML equivalent, and can be toggled between checked and unchecked states. Once the value has been set to 2, the checkbox goes into a mixed state, and from there on after, it behaves as tri-state checkbox, which toggles between checked, unchecked, and mixed (in that order).

Below is an example of XUL for both kinds of checkboxes. The first checkbox behaves normally. By setting the second checkbox's value to "2," it becomes a tristate checkbox from that moment on.

Notice that you must use the getAttribute and setAttribute AOM functions to access the value attribute. Because the checkbox is not HTML content, the value property is not supported. 

<?xml version="1.0"?> 
<?xml-stylesheet href="xul.css" type="text/css"?> 

<!DOCTYPE window> 

<window style="width: 100%; height: 100%" 
  xmlns:html="http://www.w3.org/TR/REC-html40"
  xmlns="http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul"
  onload="setTri()"> 

<html:h1>TriStateCheckbox Test 1</html:h1>

<html:script>

function setTri()
{
  var tsbox = document.getElementById("tristate"); 
              tsbox.setAttribute("value", "2");
}

</html:script>

<html:hr/>

<!-- this is a normal checkbox -->
<checkbox/>Dual state<html:br/>

<!-- this is a tristate checkbox after the setTri() 
        function has been called -->
<checkbox id="tristate"/>Tri state baby!<html:br/>

<html:button onclick="setTri()">Click Me To Set TriState
</html:button>

</window>

The checkbox's look on a mouse down inside the box can be controlled using CSS. Currently an attribute called depress is set, and this can be used to control the look on a mouse down. [This will eventually be replaced with :active.] 

checkbox {
   display:inline;
   border: 1px solid black;
}

checkbox[depress] {
   background: darkGray;
   border: 1px inset black;
}
