What are codes in Bugzilla?

Codes are utilized by various developmental engineers and QA testers to assist in the query of a group of bugs with a related issue.  These codes may appear for a short time as needed for a particular developmental period or may be used throughout the project.
UPDATE: Please read !!
Many of the code definitions used in the Summary and Status Summary fields below are slowly being moved to the new "Keywords" field now available in Bugzilla reports.  Only keywords currently active and in use will be added to the list at: http://bugzilla.mozilla.org/describekeywords.cgi.  This doc is mainly a historical reference of previous as well as a few live codes utilized by mozilla folks.

Updates for this page are indicated as follows:

  • Live codes still being used will be noted below in bold throughout entire table row.
  • Codes moved to keyword field use will be noted in italic.
  • RETIRED!! - indicates this code was use before we had a component for the area.  Please use the product/component indicated from now on, not the code.
  • Plain text means we are not sure if code is still in use :-)
