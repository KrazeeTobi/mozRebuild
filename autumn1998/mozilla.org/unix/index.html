Unix
Much of Unix's history is rooted in open-source software. The Unix version of Mozilla is an exciting, open source project that encourages all developers to help build the better browser. Most of Mozilla is now cross-platform using NGLayout and XPFE but the Unix user interface is created with GTK widgets.
