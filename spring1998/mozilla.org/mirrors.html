mirrors

The following sites have told us that they have active mirrors of our FTP site. Note that during the first few hours of a release's availability, some of these mirrors may not yet have finished mirroring.

Please try as many as possible before downloading from our FTP site directly. If you find a problem with one of the mirrors listed below (such as the linked-to directory not existing, or the server being unreachable), please tell us. (Don't worry if a site is merely a day behind us, though.)

If you operate an automated mirror site and would like to be listed here, send mail to mirrors@mozilla.org with all relevant details (geographic location, first-hop bandwidth, and frequency of mirroring.)
