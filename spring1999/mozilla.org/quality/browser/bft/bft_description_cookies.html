Basic Functional Test Case Description - Browser Cookies

Initial Conditions:

Set the Cookie preference to "Accept all Cookies" with "Warn me before accepting a cookie" checked. You need to register as a Netcenter member before you can edit the Cookie file.

Description:

To edit the Cookie file.

  1. On the Windows platform, change the name of the cookie.txt file to "old.text." On the Macintosh platform, change the name of "Magic Cookie" to "old."
  2. Launch Navigator and go to My Netscape.
  3. Click on the Preferences icon within the browser window and not from the Edit menu item list. The 'My Netscape: Preferences page will be displayed.
  4. Add or Remove the Calculator.
  5. Click the Save button.
  6. Quit and relaunch Navigator.
  7. Go to the My Netscape page.
  8. The calculator should appear or not appear depending upon your edits to the cookie file.

Link(s):

My Netscape

Expected Results:

The edits should be written to the cookie file.
