Basic Functional Test Case Description - Navigation

Initial Conditions:

Empty the disk and memory cache before you begin each test. You can find the cache section in the Advanced section of the Preferences dialog.

Description:

To use navigational commands such as Back, Forward, Reload, and Home.

Verify History/Cache Functionality

  1. Launch Navigator.
  2. Type AOL and press the enter/return key. This should display the AOL Home Page.
  3. When the AOL Home Page finishes loading, type "www.amazon.com" within the Location text area and press the enter/return key. This should display the Amazon Home Page.
  4. When the Amazon Home Page finishes loading, type "http://merchant.netscape.com/netstore/store.html" within the Location text area. This should display the Netscape NetStore.
  5. Select the Go menu.
  6. Under the Go menu, there should be a "history" list of web sites visited with a checkmark next to the currently active site.
  7. From the Go menu, select "Welcome to AOL.COM."
  8. The AOL page should load and become the active page.
  9. Close the Navigator window.

Verify Navigational Functionality

  1. Launch Navigator.
  2. Type AOL and press the enter/return key. This displays the AOL Home Page.
  3. When the AOL Home Page finishes loading, type "www.amazon.com" within the Location text area and the press the enter/return key. This displays the Amazon Home Page.
  4. When the Amazon Home Page finishes loading, type "http://merchant.netscape.com/netstore/store.html" within the Location text area and press the enter/return key. This displays the Netscape NetStore.
  5. Click on the Home Page icon on the toolbar. The home "default" page should be displayed in the browser window.
  6. From the Go menu, select Back. The browser will load the previously viewed page (http://merchant.netscape.com/netstore/store.html).
  7. Click on the Forward icon on the toolbar. The browser should reload the default home page again.
  8. Click on the Reload icon on the toolbar. The page's contents should be reloaded in the browser window.

Link(s):

AOL

Amazon

Netscape

Expected Results:

The Go menu should contain a history of previously-visited sites, with the checkmark indicating the currently-active page. Using the Back and Forward icons should send you to previously-visited sites. Clicking the Home toolbar icon should return the browser back to the "default" page. The reload toolbar icon should force the browser to "reload" the contents of the page.
