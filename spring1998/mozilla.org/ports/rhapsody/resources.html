Team Resources

Mailing lists

The development team list is moz-dev@of.org, and is archived on-line at http://www.of.org/netscape/archive/. If you would like to become a part of the team, please consider your ability to contribute along these lines:

  • Objective-C programming experience.
  • C/C++ experience; ObjC++ bridging knowledge.
  • NeXTStep/OpenStep/Yellow experience.
  • You must have a Rhapsody developer's licence.

We don't have an announcements list; please don't ask to be added to the developer's list in order to get updates. There won't be anything interesting for a while. Check these web pages for updates.

If you'd like to join the development team, please send a subscription request to mozilla@of.org. Please include some information about what you would like to work on. Similarly, you can unsubscribe by sending mail to mozilla@of.org.

The source code

Source will be managed via client/server CVS, which will be provided by mozilla.org. Details are still being worked out. The number is developers with actuall permissions to check in code will be very small, since this repository is shared with other groups. Availability of our sources to third parties will probably be in conjunction with a product release. The is no plan to provide resources such as anonymous CVS for people outside of the development group.
