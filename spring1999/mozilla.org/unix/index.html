Unix
Much of Unix's history is rooted in open-source software. The Unix version of Mozilla is an exciting, open source project that encourages all developers to help build the better browser. Mozilla has a cross-platform (XP) architecture using NGLayout and XPToolkit, and the primary UI is implemented with GTK+.

Current Status. . . . . previous status


[May 29, 1999]
  • Alexander Larsson has contributed a GTK+ widget which can be used to easily embed gecko into any gtk application. (mozilla/webshell/embed/gtk) To build it you need to add the following configure flag to your mozilla build: 
      ./configure --enable-gtk-mozilla
    this will build the library and 2 sample tests. Many thanks to Alexander Larsson for contribution this very cool widget. (Ramiro)
